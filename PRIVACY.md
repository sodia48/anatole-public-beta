# Confidentialité — Anatole Public Beta

Anatole peut conserver un identifiant pseudonymisé, les préférences, watchlists,
positions de démonstration, alertes, notifications et commentaires de bêta.

Ne saisissez jamais de mots de passe, numéros de comptes, cartes, pièces
d’identité ou clés API dans l’interface.

Les utilisateurs peuvent supprimer leurs données depuis la page Confidentialité.
